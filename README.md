# BingSearcher

Chromium extention which can automatically search Bing with random user-defined search queries to help accumulate Microsoft reward points

## Installation

1. Clone or download this repository and unzip; save somewhere safe
2. Enable Developer mode in your Chromium-based browser (Edge preferred for extra points; Chrome, Brave, Opera, etc. also work)
3. Go to the extensions page, e.g. `edge://extensions`
4. Click "Load unpacked" and select the repository folder
5. Click the extension icon to configure and start searching!
6. (Optional, recommended) Click the extension icon and pin it to your toolbar for easy access
