BingSearcher Options

Search terms pool

The searched terms will be randomly selected among this pool. Put as many searches as you want!



Search amount

The number of searches to do each session. Aim for 25 searches to maximize rewards points!


Delay time (ms)

Time to wait between searches in ms. Try to pick the lowest possible value that still searches properly. A slower computer will need a longer delay time. Default value is 4000ms.
